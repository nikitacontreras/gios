# GIOS Hello World

A minimal command-line application written in Go for iOS.

## What it implements
- Basic Go `fmt.Println` usage.
- Standard cross-compilation for iOS armv7/arm64.

## How to use
1. Navigate to this directory:
   ```bash
   cd example/hello_world
   ```
2. Build and run on your device:
   ```bash
   gios run
   ```

## Functionality
This is the simplest example to verify your toolchain is working correctly. It compiles a single static binary and executes it on the iPad.
