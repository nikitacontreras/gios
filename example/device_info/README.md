# GIOS Device Info

A command-line tool that displays basic system information using Go's `os` and `runtime` packages.

## What it implements
- Hardware architecture detection.
- Kernel name and version display.
- Go runtime environment details.

## How to use
1. Navigate to this directory:
   ```bash
   cd example/device_info
   ```
2. Build and run:
   ```bash
   gios run
   ```

## Key features
This example demonstrates how Go's standard library interacts with the iOS system to fetch environmental data.
